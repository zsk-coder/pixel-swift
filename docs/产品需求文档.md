# PixelSwift 产品需求文档 (PRD)

> **项目代号**: PixelSwift (像素快手)  
> **产品定位**: 全球最快、最简洁的在线图片处理工具  
> **版本**: v1.0  
> **更新日期**: 2026-02-08

---

## 1. 产品概述

### 1.1 产品愿景

打造一款面向全球用户的**免费在线图片处理工具**，通过极致的用户体验和SEO优化，成为用户搜索"图片转换/压缩/调整"时的首选工具。

### 1.2 目标用户画像

| 用户类型       | 描述                         | 核心需求               | 使用场景                 |
| -------------- | ---------------------------- | ---------------------- | ------------------------ |
| **普通网民**   | 非技术背景，偶尔需要处理图片 | 简单直观，无需学习成本 | 社交媒体头像、朋友圈图片 |
| **内容创作者** | 博主、自媒体、小型电商卖家   | 批量处理，质量可控     | 产品图优化、文章配图     |
| **办公白领**   | 日常办公中需要处理文档图片   | 格式兼容，快速转换     | 文档插图、邮件附件压缩   |
| **设计师**     | 专业设计人员的辅助工具       | 精确控制，多格式支持   | 快速导出不同规格的图片   |

### 1.3 核心价值主张

1. **极速体验** - 拖拽即用，3秒完成处理
2. **隐私安全** - 本地浏览器处理，图片不上传服务器
3. **完全免费** - 无水印、无限制、无需注册
4. **多语言支持** - 覆盖全球主要语言市场

## 2. 功能模块详细规格

### 2.1 通用转换器 (Universal Converter)

#### 2.1.1 功能定义

将图片从一种格式转换为另一种格式，支持批量操作。

#### 2.1.2 输入格式支持

| 格式 | 扩展名       | 说明                   |
| ---- | ------------ | ---------------------- |
| JPEG | .jpg, .jpeg  | 最常见的有损压缩格式   |
| PNG  | .png         | 支持透明通道的无损格式 |
| WebP | .webp        | Google开发的现代格式   |
| HEIC | .heic, .heif | Apple设备默认照片格式  |
| BMP  | .bmp         | Windows位图格式        |
| TIFF | .tif, .tiff  | 高质量印刷格式         |

#### 2.1.3 输出格式支持

| 格式 | 可用质量设置   | 特殊选项          |
| ---- | -------------- | ----------------- |
| JPG  | 1-100 (默认85) | 渐进式JPEG开关    |
| PNG  | 无损 (固定)    | PNG-8/PNG-24 选择 |
| WebP | 1-100 (默认80) | 有损/无损切换     |
| PDF  | 不适用         | 多图合并为单PDF   |

  - 页面尺寸: A4/Letter/原图尺寸

### 2.2 图片压缩器 (Image Compressor)

#### 2.2.1 功能定义

在保持可接受画质的前提下，最大程度减小图片文件体积。

#### 2.2.2 支持格式

| 输入格式 | 输出格式 | 说明       |
| -------- | -------- | ---------- |
| JPG      | JPG      | 保持原格式 |
| PNG      | PNG      | 保持原格式 |
| WebP     | WebP     | 保持原格式 |

#### 2.2.3 压缩控制

**质量滑块规格：**

| 属性     | 值             |
| -------- | -------------- |
| 最小值   | 0%             |
| 最大值   | 100%           |
| 默认值   | 80%            |
| 步进值   | 1%             |
| 显示格式 | 实时显示当前值 |

**压缩档位预设：**

| 档位名称 | 质量值 | 适用场景           | 预估压缩率 |
| -------- | ------ | ------------------ | ---------- |
| 极致压缩 | 30%    | 缩略图、预览图     | 80-90%     |
| 推荐压缩 | 60%    | 网页展示、社交分享 | 60-70%     |
| 轻度压缩 | 80%    | 高质量展示         | 30-50%     |
| 接近无损 | 95%    | 对质量敏感的场景   | 10-20%     |

**双栏对比模式：**

### 2.3 尺寸调整器 (Image Resizer)

#### 2.3.1 功能定义

调整图片的宽度和高度尺寸。

#### 2.3.2 支持格式

输入：任意浏览器可识别的图片格式  
输出：与输入相同

#### 2.3.3 调整模式

**模式一：按像素调整**

| 参数      | 说明           | 限制      |
| --------- | -------------- | --------- |
| 宽度 (px) | 目标宽度像素值 | 1 - 10000 |
| 高度 (px) | 目标高度像素值 | 1 - 10000 |
| 锁定比例  | 保持原始宽高比 | 默认开启  |

**模式二：按百分比调整**

| 参数     | 说明             | 限制      |
| -------- | ---------------- | --------- |
| 缩放比例 | 相对原图的百分比 | 1% - 500% |

**模式三：按宽高比裁剪**

| 预设比例 | 适用场景             |
| -------- | -------------------- |
| 1:1      | 头像、社交媒体方形图 |
| 4:3      | 传统照片、PPT        |
| 16:9     | 视频封面、网页横幅   |
| 3:2      | 单反相机照片         |
| 9:16     | 手机壁纸、短视频封面 |
| 自定义   | 用户输入任意比例     |

```

│  └─────────────────────────────────────────────────────────┘   │
│                                                                  │
│  调整方式:  ( ● ) 像素  ( ○ ) 百分比  ( ○ ) 宽高比              │
```

#### 2.3.5 锁定比例行为

当"保持宽高比"开启时：

| 用户操作 | 系统行为                             |
| -------- | ------------------------------------ |
| 修改宽度 | 自动计算并更新高度                   |
| 修改高度 | 自动计算并更新宽度                   |
| 同时修改 | 以最后修改的值为准，自动调整另一个值 |

### 4.5 响应式设计规范

| 断点名称      | 宽度范围        | 典型设备                   |
| ------------- | --------------- | -------------------------- |
| xs (手机)     | < 576px         | iPhone SE, 小屏手机        |
| sm (大屏手机) | 576px - 767px   | iPhone 14, Android主流机型 |
| md (平板)     | 768px - 1023px  | iPad Mini, iPad            |
| lg (小桌面)   | 1024px - 1279px | iPad Pro, 小笔记本         |
| xl (桌面)     | ≥ 1280px        | 标准桌面显示器             |

#### 4.5.3 移动端优化要点

**触控适配：**

| 元素       | 最小点击区域   |
| ---------- | -------------- |

| 滑块拖动区 | 高度增大至24px |
| 列表项     | 整行可点击     |

| 平台   | 上传方式                      |

| 策略     | 说明                                 |
| -------- | ------------------------------------ |

| 频次控制   | 插屏广告同一用户每日最多显示3次 |
