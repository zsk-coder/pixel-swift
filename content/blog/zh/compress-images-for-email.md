## 为什么邮件附件有大小限制

各大邮箱的附件大小限制如下：

| 邮箱服务                | 最大附件大小                    |
| ----------------------- | ------------------------------- |
| **Gmail**               | 25 MB                           |
| **Outlook / Hotmail**   | 20 MB                           |
| **Yahoo 邮箱**          | 25 MB                           |
| **Apple Mail (iCloud)** | 20 MB                           |
| **QQ 邮箱**             | 50 MB（普通）/ 2 GB（超大附件） |
| **163 邮箱**            | 50 MB                           |

这些限制是针对**所有附件的总大小**——而不是单个文件。如果你要发 5 张 5 MB 的照片，总共就是 25 MB，已经到了 Gmail 的上限。

现在的手机摄像头让这个问题越来越严重。一张 iPhone 或华为手机拍的照片轻松就有 **5-12 MB**，专业相机拍的照片则通常在 **15-30 MB**。

## 为什么不直接用网盘链接？

你可能会想："为什么不直接传到网盘然后分享链接？"

这种方式有时可行，但存在明显缺点：

- **接收者可能无法访问** —— 共享链接需要权限设置，公司防火墙经常屏蔽网盘链接
- **链接会过期** —— 网盘链接可能被撤销或到期
- **不够专业** —— 在商务沟通中，直接附件比"点击链接下载"更正式
- **离线无法查看** —— 附件可以离线查看，链接不行

图片压缩两全其美：**文件小到可以直接附加到邮件**，无需链接。

## 选择合适的图片格式

不同的图片格式在邮件场景中表现不同：

| 格式         | 最适合             | 典型压缩率  | 邮箱兼容性                |
| ------------ | ------------------ | ----------- | ------------------------- |
| **JPEG/JPG** | 照片、有渐变的截图 | 减小 60-85% | ✅ 通用                   |
| **PNG**      | 含文字的截图、Logo | 减小 30-50% | ✅ 通用                   |
| **WebP**     | 追求最大压缩       | 减小 70-90% | ⚠️ 部分旧邮箱可能无法预览 |

**邮件推荐**：照片用 **JPEG**，截图用 **PNG**。这两种格式在所有设备的所有邮箱中都能正常显示。JPEG 质量设为 75% 是最安全的选择。

需要转换格式？PixelSwift 的[图片格式转换器](/zh/converter)可以在 JPG、PNG、WebP 之间一键互转。
